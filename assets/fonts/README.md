# Fonts

Plus Jakarta Sans is used as the app's primary typeface. Font files are included in this directory and registered in `pubspec.yaml` under `flutter.fonts`.

- **PlusJakartaSans-Regular.ttf** — default weight
- **PlusJakartaSans-Medium.ttf** — weight 500
- **PlusJakartaSans-SemiBold.ttf** — weight 600
- **PlusJakartaSans-Bold.ttf** — weight 700

Source: [Google Fonts](https://fonts.google.com/specimen/Plus+Jakarta+Sans) (SIL Open Font License).
